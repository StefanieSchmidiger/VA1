This folder contains S19 (Motorola S-Record) files to recover the OpenSDA bootloader on the K20 of the Freescale/NXP FRDM-K64F board.

- FRDM-K64F_K20_mbed_0226.s19: bootloader plus the mbed 0226 firmware combined
- FRDM-K64F_K20_bootloader_0x0000-0x4FFF: bootloader only (memory range 0x0000 to 0x4FFF

For details see:
http://mcuoneclipse.com/2014/11/10/recovering-the-frdm-k64f-bootloader-or-cloning-the-program-of-a-microcontroller/
