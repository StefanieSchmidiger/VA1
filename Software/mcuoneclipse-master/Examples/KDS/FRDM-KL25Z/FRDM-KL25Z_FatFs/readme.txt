For the ADAFRUIT (https://www.adafruit.com/product/254) SD Card breakout board:
In the SD_Card component
- Card detection is LOW active: no
- Write protection: Disabled

For the Arduino Data Logger Shield:
In the SD_Card component
- Card detection is LOW active: yes
- Write protection: Enabled (PTA13)

See 
- https://mcuoneclipse.com/2016/07/09/fatfs-with-adafruit-microsd-breakout-board-and-nxp-frdm-kl25z/
- https://mcuoneclipse.com/2012/11/18/arduino-data-logger-shield-with-the-frdm-kl25z-board/
