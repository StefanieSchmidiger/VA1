This project uses the TWR-MCF52529 with Processor Expert.
It features FreeRTOS with FatFS using an custom SD card (INTRO SD Card).his i

TWR-MCF52259 Jumper settings:
- J4 installed (default)
- J3 (ClockMod1) and J6 (ClockMod0): 1-2 (default)
- J5 (ClockSel): 1-2 (default)
- J10 (XTAL) 2-3 (default)
- J7: 
     POT_ISO : removed
     LED4_ISO: removed
     LED3_ISO: removed
     LED2_ISO: installed (default)
     LED1_ISO: installed (default)
     Z_ISO   : removed
     Y_ISO   : removed
     X_ISO   : removed
- SW2: all on (default)
- J12: not installed (default)
- J12, J13 (TXD_SEL, RXD_SEL): 1-2 (default)
- J14, J15, J16: all on 1-2 (default)     
- J20 (BGLD): removed (default)
