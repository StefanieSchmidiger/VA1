readme.txt

This project is using the TWR-CN128 with the LCD Rev0 (the one without JM128).

Jumper settings:
----------------
- LED1_ISO and LED2_ISO removed
- J11/Clk_Sel: 1-2
- J12/XTAL_Sel: 1-2
- J13/Latch-Sel: 2-3
- J16/Btld: removed
- RXD_SEL and TXD_SEL: 1-2 (Route SCI to onboard connector)
- ANA_ISO: 13-14 and 11-12 set

TWR-LCDSTY-REV.0:
- BL-ISO installed
- CS-SEL: 1-2
- SPI-CS: 1-2
- PS2: installed
- PS1: removed
