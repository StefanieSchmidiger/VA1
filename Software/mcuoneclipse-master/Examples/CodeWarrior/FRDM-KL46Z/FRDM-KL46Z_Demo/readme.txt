readme.txt
==========
This is a demo project for the FRDM-KL46Z board using CodeWarrior for MCU10.6.
It uses extra Processor Expert components from the MCUonEclipse project, see
http://mcuoneclipse.com/2014/10/21/mcuoneclipse-releases-on-sourceforge/

See:
http://mcuoneclipse.com/2013/08/06/a-new-freedom-board-frdm-kl46z/

Licensing:

http://www.freescale.com/webapp/sps/site/prod_summary.jsp?code=FRDM-KL46Z&fpsp=1&tab=Design_Tools_Tab#
KL46_SC (FRDM-KL46Z and TWR-KL46Z48M Sample Code Package) from www.freescale.com/freedom
