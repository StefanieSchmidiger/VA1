Demo program with nRF24L01+ transceiver and the ElecFreak Joystick board.

See:
http://mcuoneclipse.com/2014/04/27/joystick-shield-with-the-frdm-board/

Snake Game
==========
Press Joystick button to start/pause game.
Use up(A), down(C), left(D), right(B) push buttons to move snake.
