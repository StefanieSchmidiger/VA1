readme.txt
----------
This is a SD card data logger project for the FRDM-K64F board.
The project is for Eclipse Kepler with GNU ARM Eclipse and Processor Expert.
It logs the accelerometer values to a log file on the SD card.

See this tutorial:
http://mcuoneclipse.com/2014/05/26/tutorial-data-logger-with-the-frdm-k64f-board/
