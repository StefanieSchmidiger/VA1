NeoPixel Shield:
----------------
Altium_NeoPixel.zip contains the CAD files for the NeoPixel Shield (see http://mcuoneclipse.com/2014/11/10/neoshield-ws2812-rgb-led-shield-with-dma-and-nrf24l01/)Altium_NeoPixel_Schematics.pdf contains the BOM, schematics and layout.
